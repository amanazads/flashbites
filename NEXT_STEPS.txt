🚨 IF MONGODB NOT RUNNING:
═══════════════════════════════════════════════════════════════

The default .env uses: mongodb://localhost:27017/flashbites

If you don't have local MongoDB:
1. Go to https://www.mongodb.com/cloud/atlas
2. Create free cluster
3. Get connection string
4. Edit server/.env
5. Replace MONGODB_URI with your string
